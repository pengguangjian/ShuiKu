# Installation with CocoaPods
``` 
pod 'PGBarChart'
```

# Usage
```
#import <PGBarChart.h>

PGBarChart *barChart = [[PGBarChart alloc]initWithFrame:self.view.frame];
[self.view addSubview:barChart];
```
可以下载demo参考使用方法
